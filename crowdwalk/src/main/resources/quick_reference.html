クイック・リファレンス

共通 (Background Image モードを除く)
  右クリックでドラッグ   マップのスクロール
  ホイール   マップ表示の拡大・縮小
  Ctrl + ホイール   マップ表示の拡大・縮小(1/10 ステップ)
  Alt + ホイール   編集画面の回転
  Alt + Ctrl + ホイール   編集画面の回転(1/10 ステップ)
  Ctrl + A (テーブル)   テーブルの全データを選択
  Ctrl + Z   Undo
  Ctrl + Y   Redo
  Ctrl + Q   エディタ終了
Add Node モード
  左クリック   ノード追加
Add Link モード
  左クリック   始点ノード選択 / 終点ノード選択 & リンク追加
  右クリック   ニュートラル
  ESC   ニュートラル
Add Node & Link モード
  左クリック   始点ノード追加 / 終点ノード追加 & リンク追加
  右クリック   ニュートラル
  ESC   ニュートラル
Edit Node モード
  左クリック   ノード選択
  Ctrl + 左クリック   ノードの追加選択 / 選択解除
  左クリックでドラッグ   矩形内のノードを選択
  Alt + 左クリック   ポリゴンによるノード選択
  Shift + 左クリックでドラッグ   ノードの移動
  右クリック   コンテキストメニュー表示
  Ctrl + A   カレントグループの全ノードを選択(フィルタの影響を受ける)
  ESC   選択を解除
Edit Link モード
  左クリック   リンク選択
  Ctrl + 左クリック   リンクの追加選択 / 選択解除
  左クリックでドラッグ   矩形内のリンクを選択
  Alt + 左クリック   ポリゴンによるリンク選択
  右クリック   コンテキストメニュー表示
  Ctrl + A   カレントグループの全リンクを選択(フィルタの影響を受ける)
  ESC   選択を解除
Edit Area モード
  左クリック   エリア選択
  Ctrl + 左クリック   エリアの追加選択 / 選択解除
  左クリックでドラッグ   矩形内のエリアを選択
  Alt + 左クリック   ポリゴンによるエリア選択
  右クリック   コンテキストメニュー表示
  Ctrl + A   カレントグループの全エリアを選択(フィルタの影響を受ける)
  ESC   選択を解除
Background Image モード
  右クリックでドラッグ   マップのスクロール
  Shift + 左クリックでドラッグ   背景画像の移動
  ホイール   背景画像の拡大・縮小
  Ctrl + ホイール   背景画像の拡大・縮小(1/10 ステップ)
  Alt + ホイール   背景画像の回転
  Alt + Ctrl + ホイール   背景画像の回転(1/10 ステップ)
  右クリック   コンテキストメニュー表示
